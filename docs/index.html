Finclaw

A tiny CLI that lets an agent read your bank account data through Plaid, from your machine, as JSON.

npx finclaw auth
npx finclaw accounts
npx finclaw balances
npx finclaw transactions --days 30

How It Works

1. Connect

Run npx finclaw auth. Finclaw opens Plaid Link in your browser and saves the exchanged access token locally.

2. Query

Run commands like balances or transactions. Output is JSON so agents and scripts can parse it.

3. Stay Local

Finclaw stores token metadata at ~/.finclaw/config.json and talks directly to Plaid. There is no hosted Finclaw backend.

Credentials

Create a Plaid app, then set:

export PLAID_CLIENT_ID=your-client-id
export PLAID_SECRET=your-secret
export PLAID_ENV=sandbox

For Chase and other OAuth institutions, configure a Plaid redirect URI such as:

export PLAID_REDIRECT_URI=http://localhost:7777/oauth-return
npx finclaw auth --env development --port 7777

Commands

finclaw auth
finclaw accounts
finclaw balances
finclaw transactions --start 2026-04-01 --end 2026-05-01
